## Datos

- Población total = 500000  

- nº de habitantes de la Muestra = 45% de la población total = 0.45  

- Probabilidad enfermo = 2% de la muestra = 0.02  

## Pasos

1. Resolución
```
        Python
        habitantes_muestra = 500000 * 0.45 = 225000
```       

2. Cálculo de una persona enferma

    2.1. Resolución
```
        Python
        personas_enfermas = 225000 * 0.02 = 4500
```

3. Conclusion
